# Rishikesh Bhagat — Portfolio

A modern, responsive portfolio website built with **HTML, CSS, and JavaScript**. The portfolio highlights my experience, technical skills, projects, education, and achievements as an AI Backend Developer.

![Status](https://img.shields.io/badge/status-live-brightgreen)
![Built With](https://img.shields.io/badge/built%20with-HTML%2FCSS%2FJS-blue)
![License](https://img.shields.io/badge/license-MIT-lightgrey)

---

## ✨ Features

- Modern glassmorphism-inspired UI
- Fully responsive design
- Interactive hero section with typing animation
- Animated skill progress bars
- Scroll reveal animations
- Sticky navigation with active section highlighting
- Back-to-top button
- Semantic HTML and accessibility-friendly design
- Built using pure HTML, CSS, and JavaScript

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript (ES6)**
- **Font Awesome**
- **Google Fonts**

---

## 📁 Project Structure

```text
portfolio/
│── index.html
│── README.md
```

The project is intentionally built as a **single-file portfolio**, with all HTML, CSS, and JavaScript contained inside `index.html`.

---

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Rishi-314/portfolio.git
cd portfolio
```

### Run Locally

Simply open `index.html` in your preferred web browser.

For development, you can also use the **Live Server** extension in Visual Studio Code.
